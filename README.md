# cv-interactivo
 Hola añado esta indicación, Mi CV estuve varios dias generando una funcionalidad para que en el boton de arriba d ela pagina pudiera cambiar el color completo de la pagina en html, en css y js. Pero no pude resolverlo, buscando y probando la forma en que hice que funcionara era metiendo todo en el html.
 En palabras tecnicas lo que paso fue: El problema es que Bootstrap está sobrescribiendo los estilos. Asique genere "!important" en TODOS los estilos del modo oscuro así jQuery lo aplica a la clase correctamente.
